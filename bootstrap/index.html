HTML CSS & Javascript

Bootstrap uses components from html, css, and javascript to style webpages. Once you've downloaded bootstrap you can use classes (a full list of which can be found on w3 schools) to target and style components of your webpage.

Responsivity

Bootstrap it automatically responsive to different screen sizes. This means the style will change depending on the device you are using

Components

Bootstrap uses a large array of components to help style your webpage. You can style everything from the text to navigation to buttons

buttons

You can style buttons too look a couple of different ways.
